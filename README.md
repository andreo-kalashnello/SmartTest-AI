# SmartTest AI

[![Next.js](https://img.shields.io/badge/Next.js-000000?style=flat&logo=next.js&logoColor=white)](https://nextjs.org/)
[![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=flat&logo=typescript&logoColor=white)](https://www.typescriptlang.org/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind-06B6D4?style=flat&logo=tailwindcss&logoColor=white)](https://tailwindcss.com/)
[![PostgreSQL](https://img.shields.io/badge/PostgreSQL-4169E1?style=flat&logo=postgresql&logoColor=white)](https://www.postgresql.org/)

**SmartTest AI** — платформа для образовательного процесса: создание, проведение и анализ тестов с поддержкой ИИ. Два типа пользователей — **преподаватель** (контент, настройки, аналитика) и **ученик** (прохождение без лишней регистрации, по PIN).

На этапе **тестовой v1** приоритет — рабочий сценарий «преподаватель → тест → ученик → результат» и документация для сдачи; полный функционал из исходного ТЗ — **к полному релизу (~июнь)**.

## Фокус разработки

Команда на старте — **ориентирована на фронтенд**: основной объём для тестовой v1 — **интерфейс преподавателя, экран ученика, прохождение теста, базовая интеграция с API**. Бэкенд — минимально необходимое подмножество эндпоинтов и схемы БД под этот сценарий.

---

## Архитектура

Монорепозиторий на **pnpm**: каталог [`web/`](web/) — Next.js (App Router) и **FSD-слои** в [`web/src`](web/src) (`app`, `views`, `widgets`, `features`, `entities`, `shared`). Слой экранов назван **`views`**, чтобы не использовать каталог `src/pages` (он в Next.js резервируется под [Pages Router](https://nextjs.org/docs/pages)). Каталог [`api/`](api/) зарезервирован под бэкенд (пока пустой, см. [`api/README.md`](api/README.md)).

Приложение в целом — **Next.js fullstack** с разделением UI ↔ API ↔ БД; документированные чеклисты остаются источником правды по функционалу.

## Быстрый старт

Требования: **Node.js 20+**, **pnpm** (рекомендуется 9+).

```bash
pnpm install
pnpm dev
```

Откройте [http://localhost:3000](http://localhost:3000) — тестовая главная в [`web/src/views/home`](web/src/views/home). Переменные для фронта: скопируйте [`web/.env.example`](web/.env.example) в `web/.env.local` при необходимости.

Сборка фронта: `pnpm build`. Бэкенд в `api/` подключается отдельно, когда будет готов.
