# Чеклист: сторінки фронтенду (SmartTest AI)

Метки: **v1-test** — для тестової версії; **june-full** — до повного релізу.
Маршрути відповідають App Router у `web/src/app/`.

---

## Публічні сторінки

| Сторінка | Маршрут | Статус |
|----------|---------|--------|
| Лендінг (головна) | `/` | **v1-test** |
| Реєстрація | `/register` | **v1-test** |
| Вхід | `/login` | **v1-test** |
| Вхід учня за PIN | `/join` | **v1-test** |
| 404 Not Found | — | **v1-test** |

### Чекбокси

- [x] **v1-test** `/` — лендінг зі шапкою, Hero, Можливості, Як це працює, Для кого, CTA, підвал
- [ ] **v1-test** `/register` — форма реєстрації (email, пароль, ім'я) для викладача
- [ ] **v1-test** `/login` — форма входу (email + пароль), посилання на реєстрацію
- [ ] **v1-test** `/join` — форма учня: PIN + ПІБ, перехід до плеєра
- [ ] **june-full** `/forgot-password` — відновлення пароля
- [ ] **june-full** `/verify-email` — підтвердження пошти

## Definition of Done — тестова v1 (сторінки)

1. Усі маршрути `v1-test` відображаються без помилок 404/500.
2. Сторінки реєстрації та входу ведуть на дашборд після успішного запиту.
3. Учень за PIN потрапляє в плеєр, проходить тест, бачить результат.
4. Дашборд захищений — неавторизований користувач перенаправляється на `/login`.
5. Усі сторінки адаптивні (mobile-first).
