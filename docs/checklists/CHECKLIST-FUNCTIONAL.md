# Чеклист: функционал продукта (по исходному ТЗ)

## 1. Блок преподавателя

### 1.1 AI-генератор тестов

- [x] **v1-test** Упрощённый сценарий: тема + количество + сложность + текст материала → `/api/ai/create-test`
- [ ] **june-full** Загрузка PDF, DOCX, TXT
- [ ] **june-full** «Умные» дистракторы, тонкая настройка промптов

### 1.2 Редактор тестов (manual)

- [x] **v1-test** CRUD вопросов и вариантов, один правильный ответ
- [ ] **june-full** Несколько правильных, true/false, медиа, банк

### 1.3 Настройки тестирования

- [x] **v1-test** Завершение теста и простой результат ученику
- [ ] **june-full** Таймер, античит, окно доступности, настройки feedback

### 1.4 Дистрибуция

- [x] **v1-test** Уникальный PIN (6 цифр), ссылка `/join` и `/test/[pin]/play`
- [ ] **june-full** QR, PDF-бланки

### 1.5 Аналитика

- [x] **v1-test** Список попыток: ФИО, балл, время
- [ ] **june-full** Live, средний балл, экспорт, сложные вопросы
